A. Add a new exported pure helper (data filtering only, no network calls):
   export function extractFootballLeagueIds(fanData: unknown, season: number): string[]
   - Read fanData.preferences if it is an array; otherwise return [].
   - Keep preferences where typeId === 9 AND metaData.entry.abbrev === "FFL"
     AND metaData.entry.seasonId === season.
   - Collect String(groupId) from every metaData.entry.groups[] item with a
     non-null groupId.
   - De-duplicate and return. Use optional chaining throughout so malformed
     items are skipped, never crash.

     (curly braces stay in).
